== Description ==

Bottom Bar is a WordPress plugin which includes a several features (you can see them below). The idea for creating this plugin comes from the facebook bar *(in old version; now there is only a chat bar)* and from wibiya's toolbar.

**Features**

* Recent posts
* Popular posts
* Recent comments
* Random post
* Share options *(delicious, digg, edno23, facebook, google bookmarks, google buzz, svejo, twitter, stumbleupon, reddit)*
* Language packs *(english, bulgarian, deutsch, french, russian, spanish, czech)*
* Restore option *(admin option)*

**IMPORTANT:** The plugin requires **wp_head()** and **wp_footer()** in order to work properly because they makes some calls to make the plugin appear.perly.

== Installation ==

1. Unpack the download-package.
2. Upload 'bottom-bar' folder to the '/wp-content/plugins/' directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Done and Ready.

[More information](http://www.svil4ok.com/wordpress-plugins/bottom-bar/#installation)
